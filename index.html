Resumen profesional

*Desarrollo

Mi logro actual más destacado es la creación de un proyecto personal enfocado en la predicción de precipitaciones a partir de datos estadísticos tomados de la estación meteorológica de la Universidad de Cundinamarca seccional Ubaté donde a partir de la creación de un modelo denominado Modelo Comparativo de predicción de Precipitaciones(MCPP) se comparan modelos de machine learning como Naive Bayes, Maquina Vector Soporte y Random Forest buscando generar la predicción más acertada frente a cuando ocurrirá una lluvia, Este proyecto ya fue aprobado para dos ponencias, se encuentra en proceso de validación para la publicación de un artículo en la revista científica Respuestas, se encuentra en proceso de registro de software, y Fue ganador al mejor proyecto de gestión del conocimiento en la Universidad de Cundinamarca seccional Ubaté.
